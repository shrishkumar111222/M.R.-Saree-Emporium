# M.R. Saree Emporium — Website

Premium e-commerce & B2B wholesale website for **M.R. Saree Emporium**, Kohinoor Textile Market, Ring Road, Surat.

## Current status: Homepage prototype (Phase 1)

`index.html` is a fully responsive, self-contained homepage — no build step required. Open it in any browser, or view it live via GitHub Pages.

### What's included

- Premium Indian luxury-textile design (wine / ivory / muted gold, Fraunces + Karla typography)
- Sticky header, announcement bar, mobile slide-out menu with expandable categories
- Cinematic hero with entrance animations
- Asymmetric editorial category grid (8 categories)
- New Arrivals slider and Best Sellers grid with sample products
- Wholesale / B2B section with 6-step ordering flow
- Shop by Occasion and Shop by Fabric
- Animated business stats, brand story, testimonials (sample placeholders)
- Store section with real address, hours, phone, and embedded Google Map
- WhatsApp commerce: floating button + pre-filled product enquiry messages to +91 97272 36377
- LocalBusiness schema markup, Open Graph tags, semantic HTML, reduced-motion support

## Roadmap (Phases 2+)

Full platform per the project spec: Next.js + TypeScript + Tailwind, PostgreSQL + Prisma, role-based auth (Customer / Wholesale Buyer / Staff / Admin), wholesale approval workflow, cart & Razorpay-ready checkout, admin dashboard & CMS, product catalogue with filtering, SEO & performance targets (Lighthouse 90+).

## Viewing live

Enable **GitHub Pages** (Settings → Pages → Deploy from branch → `main` → root) and the homepage will be served at the repository's Pages URL.
